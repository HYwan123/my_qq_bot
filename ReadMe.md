# 当前系统规划(未实现的)

## 数据存储与处理

- **Redis**
  - 用途：存储用户简介（用户元信息、偏好等）
  - 特性：快速访问，适合短期和频繁读取的数据

- **PostgreSQL**
  - 用途：存储用户聊天记录
  - 特性：关系型数据库，支持复杂查询和事务

- **Milvus**
  - 用途：存储知识库（向量化文档、嵌入等）
  - 特性：高效向量检索，适合语义搜索和知识问答

## 系统功能模块

- **超时归档**
  - 说明：定期将过期或长时间不活跃的数据归档处理，压缩上下文

- **MCP 服务**
  - 功能：
    - 联网搜索（获取实时信息或外部数据）
    - 虚拟终端（执行命令或操作系统级任务）
